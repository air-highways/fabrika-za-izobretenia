## ?? Описание
Това репо съдържа **Claim Sheet** – документ с уникални проектни претенции, разработени в рамките на **„Клуб България: Фабрика за изобретения“**.  
Документът описва основните глобални проекти и тяхната оригиналност.

## ?? Проекти
1. **Холографчо** – холографски компютър, комбиниращ светлина, вода и многопластови структури за визуализация и изчисления.  
2. **Въздушни магистрали** – нова транспортна система с повдигнати П-образни конструкции, позволяваща повдигане на стандартни автомобили.  
3. **Български суперкар** – модулен автомобил, сглобяем от клиента, със сложни модули, предварително произведени във фабрика.  
4. **Български сериен автомобил** – достъпна основа (шаси + купе), която клиентът персонализира и адаптира към двигател по избор.
